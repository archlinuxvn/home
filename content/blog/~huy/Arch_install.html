Điều này có nghĩa là việc cài đặt Arch sẽ giống với `Gentoo` và phải có internet. -Điều này khá phiền phức cho các bạn mới đến. Vậy tại sao không dùng những gì mà chính file iso đó cung cấp cho ta ? ## Hiện thực ý tưởng - Sau khi bạn boot vào bằng iso file, sẽ cần thực hiện các bước sau: + Phân chia lại lãnh thổ cho từng vùng: /, /home, /boot và swap: fdisk /dev/sda: nhấn m để xem menu hướng dẫn. - Tôi thực hiện phân vùng theo thứ tự sau cho dễ nhớ: .Boot -> Root -> swap ->Home (sda1 -> sda2 -> sda3 ->sda4) (ext2 -> ext4 -> swap -> ext4) + `Mount` phân vùng `Root`: mount /dev/sda2 /mnt + Tai đây ta sẽ tiến hành copy toàn bộ image của hệ thống hiện tại vào /mnt: cp -R /usr /mnt cp -R /bin /mnt cp -R /sbin /mnt cp -R /etc /mnt cd /mnt mkdir dev mnt proc run srv tmp var media opt root sys home boot ln -s usr/lib lib + Vấn đề tiếp theo là coppy file `vmlinuz` và tạo `initramfs.img`: mount /dev/sda1 /mnt/boot Tiến hành copy file `vmlinuz` và `initramfs`: mount /dev/sdbX /opt cp /opt/arch/boot/i686/vmlinuz /boot cp /opt/arch/boot/i686/archiso.img /boot/initramfs.img **i686 hoặc x86_64 là do tùy thuộc bạn muốn cài hệ điều hành 32 hay 64 bit.
